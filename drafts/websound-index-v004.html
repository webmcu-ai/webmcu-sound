Browser mic recording = captures 1 second via Web Audio API → computes Mel spectrogram → adds to training buffer. Auto-saves .f32 to PC folder if set.
Device recording = sends RECORD_START to ESP32 → device captures 1s via PDM mic → saves WAV to SD card → COLLECT_COUNT updates counter.
